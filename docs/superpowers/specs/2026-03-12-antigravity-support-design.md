# Google Antigravity Support — Design Spec

**Date:** 2026-03-12
**Status:** Approved

## Summary

Add Google Antigravity as a new output format in context-gen. Antigravity is Google's agent-first AI IDE (released Nov 2025) that reads project context from `.gemini/GEMINI.md`.

## Output

- **File path:** `.gemini/GEMINI.md`
- **Format:** Plain markdown with `##` section headings (no frontmatter)
- **CLI flag value:** `antigravity`

## GEMINI.md Structure

```markdown
# GEMINI.md - {ProjectName}

{Description}

## Tech Stack
## Common Commands
## Code Style
## Project Structure
## Key Files
## Conventions
## Coding Guidelines
## Testing
## Workflow        <!-- placeholder -->
## Architecture   <!-- placeholder -->

<!-- Generated by context-gen vX.Y.Z -->
```

This mirrors CLAUDE.md (includes Workflow/Architecture placeholders) because Antigravity is an agent-first IDE that benefits from rich project context.

## Changes Required

### 1. generator/generator.go
- Add `FormatAntigravity Format = "antigravity"` constant
- Add `case FormatAntigravity` in `Generate()` → calls `generateAntigravity()`
- Add `.gemini/GEMINI.md` to `FormatAll` case
- `generateAntigravity()`: header `# GEMINI.md - {name}`, all standard sections, Workflow + Architecture placeholders

### 2. cmd/root.go
- Add `"antigravity"` case in `parseCommonFlags()` format switch
- Add to error message format list
- Add `.gemini/GEMINI.md` detection in `detectExistingFormats()`
- Add to `printUsage()` help text

### 3. cmd/interactive.go
- Add "Antigravity (.gemini/GEMINI.md)" to `promptFormat()` fallback menu
- Update choice range (1-8 → 1-9) and switch cases
- Renumber "Both" and "All" options

### 4. ui/prompts.go
- Add "Antigravity (.gemini/GEMINI.md)" to `formatOptions` slice
- Add case in `SelectedFormat()` returning `"antigravity"`
- Renumber "Both" and "All" cases

### 5. generator/generator_test.go
- `TestGenerate_AntigravityFormat`: verify file key, header, sections, placeholders
- `TestGenerate_AllFormat`: add `.gemini/GEMINI.md` to expected keys
- `TestVersionStamp_AllFormats`: automatically covered

## Design Decisions

1. **`.gemini/GEMINI.md` over `GEMINI.md`**: Antigravity's native convention; Gemini CLI also finds it via parent-directory search
2. **Dedicated generator over reuse**: Antigravity is agent-first (like Claude), needs placeholder sections unlike Cursor/Windsurf
3. **Header format `# GEMINI.md - {name}`**: Follows AGENTS.md pattern for consistency
